Hello Bootstrap

트위터 디자이너들이 만든 프레임워크로서, 업그레이드가 활발하게 일어나고 있는 오픈소스이다.

색상 컨셉이 가장 중요하다. 역할 별로 색상을 나눔에 주목한다.

1. 중요도에 따른 구분
primary: 제일 중요한 대상에게 부여하는 색상
secondary: 중요하지만 덜 중요한 대상에게 부여하는 색상

2. 위험도에 따른 구분
success: 성공적인 결과가 나오도록 하기 위한 기능. 가입 버튼 등. 항상 초록색인 건 아니므로 주의한다.
danger: 반드시 확인해야 하는 위험한 기능을 의미한다. 취소 버튼 등
warning: 사용할 때 주의를 기울여야 하는 기능을 의미한다.
info: 사용자에게 정보를 주기 위한 기능을 의미한다.

3. 주/야간에 따른 구분
light: 주간 모드, 밝은 화면
dark: 야간 모드, 어두운 화면

각종 디자인 키워드

각종 디자인을 클래스 키워드로 지정할 수 있도록 해놨음.

1. 종류
bg = background = 배경 속성
text = text 속성
p = padding 속성

- pt: padding-top 속성
- ps: padding-left 속성
- pe: padding-right 속성
- pb: padding-bottom 속성
- px: padding-left + padding-top 속성
- py: padding-top + padding-right 속성

m = margin 속성

- mt: margin-top 속성
- ms: margin-left 속성
- me: margin-right 속성
- mb: margin-bottom 속성
- mx: margin-left + margin-top 속성
- my: margin-top + margin-right 속성

.container: 너비에 따라 바로바로 변하는 외부 영역

.container-fluid: 너비에 맞게 부드럽게 변하는 외부 영역

.container: 너비에 따라 바로바로 변하는 외부 영역

.container-fluid: 너비에 맞게 부드럽게 변하는 외부 영역

한쪽이 light면 한쪽은 dark로 되도록 디자인하는 것이 좋다.
